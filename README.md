# intern-aiproject

## day3-todo

Simple command-line to-do app that stores tasks in `day3-todo/tasks.json`.

### Setup

```bash
cd day3-todo
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Usage

```bash
python todo.py add "Buy groceries"
python todo.py list
python todo.py remove 1
```
